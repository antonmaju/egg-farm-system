Overview

Adding New Additional Data

To add a new additional data/cost entry, press the New button in "Additional Data" list view.

Fill the entry with the additional data info then press the save button.

Editing Additional Data/Cost Entry

To edit an additional data/cost entry double click the entry from the list which you want to edit.

Deleting Additional Data/Cost Entry

To delete additional data/cost entry, select(single click) the additional data/cost entry from the list then press the delete button.
