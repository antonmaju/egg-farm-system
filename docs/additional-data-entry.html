Overview

To manage additional costs data, you will need to use additional costs list screen.
Additional costs list screen is located inside master data menu and under additional costs tab

Adding New Additional Data

To add a new additional cost data, press the "New" button in additional costs list screen.

Fill the entry with the additional data info then press the "Save" button.

Editing Additional Cost Data

To edit an additional cost data, just double click the corresponding data in the additional costs list screen.

Deleting Additional Data/Cost Entry

To delete additional cost data, select the corresponding data in the additional cost list screen then press the "Delete" button.
