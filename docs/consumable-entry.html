Add a new Consumable
To add a new consumable entry press the add button.

Fill the entry with the consumable detail then press the save button.


Edit Consumable Entry
To edit a consumable entry double click the entry from the list which you want to edit.

Edit the detail then press the save button.


Delete Consumable Entry
To delete consumable entry, select(single click) the consumable entry from the list then press the delete button.
