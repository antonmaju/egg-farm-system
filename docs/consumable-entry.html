Overview

To manage Consumable data, you will need to use consumable list screen.
Consumable list screen is located inside master data menu and under consumable tab

Adding New Consumable

To add a new consumable data, press the "New" button.

Fill the input fields and ensure there is no warning sign.
Click "Save" button once you've finished editing

Editing Consumable Data

To edit a consumable data, just double click the corresponding consumable from the consumable list screen.

Deleting Consumable Data

To delete a consumable data, please select the corresponding consumable from the list then press the "Delete" button.
