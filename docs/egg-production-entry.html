Overview

To manage egg production entry, you will need to use egg production screen.
Egg production screen is located in main menu navigation under egg production tab

Add a new Egg Production

To add a new egg production entry press the add button.

To add details of egg production, pick the date, fill the Good egg count(grain), Retail quantity(kg) and cracked egg count(grain) on each of the hen house, then press the save button.

Edit Egg Production Entry

To edit an egg production entry double click the entry from the list which you want to edit.

Edit the detail then press the save button.

Delete Egg Production Entry

To delete an egg production entry, select(single click) the egg production entry from the list then press the delete button.
