Overview

Egg production list screen is used for managing daily egg production.
It can be accessed from Egg Production main menu

Add a new Egg Production

To add a new egg production entry, you will need to press the New button in egg production list screen.
You will enter egg production entry screen after that.

The egg production entry consists of several important fields:

  • Date

    The date field is represented by a date picker in the middle. Bear in mind that there could only be 1 date for every egg production entry.

  • House

    The hen house which the egg produced.

  • Good Egg Count

    The count/amount of good egg.

  • Retail Quantity

    The weight of good egg (in kilogram).

  • Cracked Egg Count

    The count/amount of cracked egg.

To add details of egg production, pick the date, fill the Good egg count, Retail quantity and cracked egg count on each of the hen house.Once it's complete press the save button.

Edit Egg Production Entry

To edit an egg production data, you will need to be in egg production list screen.
Just double click the corresponding egg production data to continue to egg production entry screen.

Edit the detail, once it's complete press the save button.

Delete Egg Production Entry

To delete an egg production data, you will need to be in egg production list screen.
You will need to select the egg production data from the list, then press the delete button.
