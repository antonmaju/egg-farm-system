Add a new Employee
To add a new employee entry press the add button.

Fill the entry with the employee detail then press the save button.


Edit Employee Entry
To edit an employee entry double click the entry from the list which you want to edit.

Edit the detail then press the save button.


Delete Employee Entry
To delete employee entry, select(single click) the employee entry from the list then press the delete button.
