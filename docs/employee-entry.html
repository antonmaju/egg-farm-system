Adding New Employee

To add a new employee, click the New button.

Fill the input fields and ensure there is no warning sign. Click Save button once you've finished editing.

Edit Employee Data

To edit an employee data, just double click the corresponding data in the employee list view.

Deleting Employee Data

To delete an employee data, please select the corresponding employee in list view then click the delete button.
