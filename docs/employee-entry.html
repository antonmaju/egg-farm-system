Overview

To manage employee data, you will need to use employee list screen.
Employee list screen is located inside master data menu and under employee tab

Adding New Employee

To add a new employee data, click the "New" button.

Fill the input fields and ensure there is no warning sign.
Click "Save" button once you've finished editing.

Editing Employee Data

To edit an employee data, just double click the corresponding data in the employee list screen.

Deleting Employee Data

To delete an employee data, please select the corresponding data in employee list screen then click the "Delete" button.
