Add a new Hen
To add a new hen entry press the add button.

Fill the entry with the hen detail, fill it correctly and no warning sign displayed then press the save button.


Edit Hen Entry
To edit a hen entry double click the entry from the list which you want to edit.

Edit the detail then press the save button.


Delete Hen Entry
To delete hen entry, select(single click) the hen entry from the list then press the delete button.
