Adding a new Hen

To add a new hen entry please press the "New" button in hen list view.

Please fill the fields with the hen detail and ensure there is no warning sign.
Press the save button once you've finished editing.

Editing Hen Data

To edit a hen data, please double click the corresponding item in hen list.

Edit the detail then press the save button.

Deleting Hen Data

To delete a hen entry, select(single click) the hen entry from the list then press the delete button.
