Add a new Employee Cost
To add a new employee cost entry press the add button.

To add details of employee cost, pick the date, tick/untick the 'present' checkbox to add/not add employee info to employee cost detail, set the salary and description if needed, then press the save button.


Edit Employee Cost Entry
To edit an employee cost entry double click the entry from the list which you want to edit.
(note: you can use pager on below the list to switch page)

Edit the detail then press the save button.


Delete Employee Cost Entry
To delete employee cost entry, select(single click) the employee cost entry from the list then press the delete button.
