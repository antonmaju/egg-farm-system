Add a new Consumable Usage
To add a new consumable usage entry press the add button.

To fill the details of Consumable usage, pick the date, select hen house and consumable, fill the ammount of count and press enter to count the subtotal. If the detail is complete press the save button.

Add a new row/item on details To add a row/item in details of Consumable usage, press the new button and fill the details.

Delete row/item on details To delete row/item in details of consumable usage, select(single click) the item and press the delete button.


Edit Consumable Usage Entry
To edit consumable usage entry double click the entry from the list which you want to edit.
(note: you can use pager on below the list to switch page)

Edit the detail then press the save button.


Delete Consumable Usage Entry
To delete consumable usage entry, select(single click) the consumable usage entry from the list then press the delete button.
