Overview

Consumable usage list screen is used for managing daily cost of consumable.
It can be accessed from usage main menu

Add a New Consumable Usage

To add a new consumable usage entry, you will need to press the "New" button in consumable usage list screen.
You will enter consumable usage entry screen after that.

The consumable usage entry consists of several important fields:

  • Date

    The date field is represented by a date picker in the middle. Bear in mind that there could only be 1 date for every consumable usage entry.

  • House

    The hen house which the consumable will be used.

  • Consumable

    The type of consumable that will be used.

  • Count

    The count/amount of consumable.

To fill the details of Consumable usage, pick the date, select hen house and consumable, fill the amount of count and press enter to count the subtotal. Once it's complete press the "Save" button.

Add a new row/item on details

To add a row/item in consumable usage entry list, press the "New" button and fill the details.

Delete row/item on details

To delete row/item in consumable usage entry list, select the item and press the "Delete" button.

Edit Consumable Usage Entry

To edit a consumable usage data, you will need to be in consumable usage list screen.
Just double click the corresponding consumable usage data to continue to consumable usage entry screen.
(note: you can use pager on below the list to switch page)

Delete Consumable Usage Entry

To delete a consumable usage data, you will need to be in consumable usage list screen.
You will need to select the consumable usage data from the list, then press the "Delete" button.
